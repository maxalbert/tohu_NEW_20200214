    SeedGenerator

    The SeedGenerator is is used internally by tohu to produce seeds with which to reset other generators (for example, the constituent generators in a CustomGenerator instance).

    It produces integers in the range from 0 to 2^32-1, as illustrated below.

    from tohu.base import SeedGenerator

    seed_generator = SeedGenerator()

    seed_generator.reset(12345)
for _ in range(10):
    print(next(seed_generator))
